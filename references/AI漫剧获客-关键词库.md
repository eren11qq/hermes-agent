# AI漫剧获客 — 关键词库与搜索策略

> 数据来源：B站搜索实测 + Web搜索交叉验证 + 招聘/融资公开信息
> 更新日期：2026-07-23

---

## 核心发现

**社媒泛关键词无效。** 在B站搜"AI漫剧""AI动画制作""ComfyUI工作流"，返回结果 90% 是教程号和个人创作者，不是目标企业客户。

**有效策略：三层过滤法**
1. 第一层：用精准关键词在 招聘平台/融资新闻/社媒 发现公司名
2. 第二层：拿到公司名后用 ENScan_GO 硬查工商
3. 第三层：社媒信息走 enterprise-filter 软判断企业vs个人

## 二、Tier 2 关键词（社媒内容发现 — 需配合过滤）

这些关键词在 B站/小红书/抖音 使用，但必须配合 enterprise-filter 过滤掉个人创作者。

### B站 (bilibili.com)

| 关键词 | 效果 | 为什么有效 | 注意事项 |
|--------|------|-----------|---------|
| **AI漫剧 第** | 高 | 搜索带集数标记的连载内容=制作团队 | 避开"教程""入门"结果 |
| **AI动态漫 工作室** | 高 | "工作室"关键词天然过滤个人 | 测试返回了一些疑似团队号 |
| **AI漫剧 商务合作** | 中高 | 商业意图明确 | 结果较少，但质量高 |
| **AI动态漫 原创** | 中 | 原创标签=内容生产者不是教程号 | 仍有个人创作者混入 |
| **AI漫剧 IP** | 中 | IP开发=商业团队 | 找到IP方和制作方 |
| **即梦AI 漫剧** | 中 | 特定工具用户=实际生产者 | 结合工具名过滤 |

| **AI动画制作** | 最低 | 几乎全是教程 | 不推荐单独使用 |

**B站最佳搜索策略**：
```

要搜：AI漫剧 第X集、AI动态漫 工作室、AI漫剧 原创、AI漫剧 商务合作
进阶：搜到UP主→点进主页→看简介有没有"工作室/团队/商务合作"
```

## 四、已验证的高价值搜索链路

基于本轮实测，以下链路可以直接产生可用的企业线索：

### 链路 1：招聘反查（最直接）
```
搜索 "AI漫剧 招聘" → 找到招聘帖 → 提取公司名
  → ENScan_GO 查工商 → 确认企业实体
  → 社媒搜公司名 → 了解内容产出
  → enterprise-filter 打分 → 入库/过滤
```
**本轮实测产出**：西安启昱科技、浙江乐播动漫、奥睿森（北京）、北京点光文化、辽宁川映文化、漫格影视 等13家公司

### 链路 2：融资追踪（最高质量）
```
搜索 "AI漫剧 融资 2026" → 36氪/投资界/东方财富 报道
  → 提取公司名 + 融资金额 + 投资方
  → ENScan_GO 验证 → 直接标记 HOT LEAD（有钱+在扩张）
  → 查招聘动态（融资后大概率扩招）→ 触达时机最佳
```
**本轮实测产出**：星迹互动、国芯智造、灵境AI、星兴万物、八点八数字、灵漫快创、巨日禄 等7家已融资公司

### 链路 3：社媒内容发现（补充长尾）
```
B站搜索 "AI漫剧 第" "AI动态漫 工作室"
  → 找连载中的AI漫剧内容
  → 进入UP主主页 → 看简介/认证/商务合作方式
  → enterprise-filter 判断企业vs个人
  → 企业 → 进入深度调研管线
```
**本轮实测产出**：多个疑似团队账号

### 链路 4：平台生态对接（规模化）
```
关注平台官方创作者合作计划：
  - 阅文 "创作合伙人计划"（10万IP + 亿元基金）
  - 商汤 Seko AI（1300+企业客户）
  - 快看漫画（13000+ IP + 12万创作者）
  - 七猫（大规模招聘100+ AI漫剧人才）
  → 这些平台的企业客户 = 目标客户池
```
